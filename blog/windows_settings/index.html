【赛博扫盲】Windows 重装指北

15 Feb 2026 13 min read

[!info] 本文提供 Windows 11 重装系统后个人的配置分享，不构成建议，亦不提供「怎么重装系统」的教程。

由于经常在自己的电脑上瞎搞，重装系统成为了我的家常便饭，本文记录我重装系统后会进行的基本设置以及一些软件清单，同时可能会是一篇软件分享。

备份清单

应至少备份本机的无线网络驱动到非C盘或移动存储设备，以防止重装系统后因驱动原因导致无法访问网络。无线网络驱动以及其他各类驱动可在各OEM厂商的官网售后界面查询下载。

对于特殊网络环境，应该备份个人的网络环境设置到非C盘或移动存储设备，以防止重装系统后网络环境不通畅造成不必要麻烦。

如想要备份旧系统下所有驱动文件，可以管理员身份运行 PowerShell，并运行如下备份指令：

mkdir D:\DriveBackup
dism /online /export-driver /destination:"D:\DriverBackup"

该指令可将当前系统的驱动备份到 D:\DriverBackup 文件夹，亦可以自行更换路径，重装系统后，通过 Windows + S 搜索「设备管理器 (Device Manager) 」并进入，点击任意一个未安装驱动的设备，右键选中「更新驱动程序」并选择备份文件夹，点击下一步即可恢复所有驱动备份。详细教程可查阅B站专栏。

此外，请务必备份个人重要文件，笔者个人建议平时使用 Windows 操作系统时至少分一个盘符用来专门存储文件。

激活系统

如进入 Windows 过程中被强制要求登录微软账户，而因网络不通畅或网卡驱动未加载等原因遇到困难，可以按下 Shift + F10 呼出命令提示符并输入以下指令：

net user your_user_name /add
net localgroup Administrators your_user_name /add
cd OOBE
msoobe && shutdown -r

重启后即可以本地账户登录。详细教程可查阅CSDN。

如果您因不可抗力暂无能力支付正版 Windows 授权费用，请访问神秘链接激活。

[!important] 笔者坚决支持购入正版 Windows，这里仅提供另一种选择，需用户自行承担风险，如有问题概不负责！支持正版，人人有责！

系统必要设置

Windows Explorer 设置

  • 打开任意文件夹，在三点菜单$\rightarrow$选项页面设置每次打开新 Explorer 窗口进入的界面；在查看$\rightarrow$显示页面设置查看文件扩展名，酌情选择是否查看隐藏文件（对于开发者，建议开启）
  • 按下 Windows + X，选择以管理身份打开终端（老版本中为 PowerShell），输入 reg add "HKCU\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InprocServer32" /f /ve 以恢复旧版 Windows 右键菜单。

个性化设置

在桌面右键并进入相关个性化设置，有以下选项推荐设置：

  • 任务栏设置
    • 开启自动隐藏任务栏，以增大桌面可用空间，同时对于 OLED 屏幕用户减少了烧屏风险
    • 酌情隐藏「搜索」、「任务视图」、「小组件」等牛皮癣图标。
    • 将任务托盘的「触摸键盘」选择为「总是开启」，在键盘不在手边时这将是非常实用的功能
  • 主题$\rightarrow$桌面图标设计：酌情选择需要开启的图标，对于不熟悉电脑操作的用户，建议开启「此电脑」。
  • 背景设置、鼠标图标设置等，酌情选择。

电脑管家设置

对于使用微软官方纯净版镜像的笔记本用户，建议下回厂商官方的电脑管家 App（如 MyAsus、联想电脑管家），电脑管家提供了驱动自动下载、硬件设置等必要功能，以笔者的电脑为例，在 MyAsus 软件，可以设置屏幕色彩模式、独显直连、会议降噪麦克风、风扇转速等功能。请至少通过电脑管家等渠道确保自己开启了应享受的功能，例如 120Hz 刷新率！

Chrome 浏览器插件推荐

实话实说，现代用户用 PC 做工作，有 90% 的时间是花费在浏览器的，Chrome 浏览器或其他 Chromium 浏览器有着庞大的插件生态用于加强网上冲浪体验，这里介绍几款笔者自用还不错的浏览器插件。

需要注意的是，Chrome 应用商店访问困难，但仍可以通过第三方网站下载对应的 crx 文件进行安装。

  • Tempermonkey 用户脚本管理，需要在插件设置里额外开启 Allow User Scripts 获取完整权限，推荐搭配 Greasy Fork 获取脚本
  • uBlock Origin Lite 广告拦截，对于 FireFox 等仍支持 Manifest V2 标准的插件来说，亦可以使用 uBlock Origin 原版
  • Transy / 沉浸式翻译 一键翻译文档、外语视频双语字幕
  • GitZip 支持在 GitHub 下载单独文件夹
  • NextAI Translator AI 翻译（更推荐桌面版）
  • 青柠起始页
  • iCloud Bookmarks 与 Safari 浏览器同步
  • …

另外说一句，笔者使用的 Chrome 主题为 Colors

个人建议

作为普通用户，安装并设置好以上软件已经可以获得不错的电脑使用体验，当然前提是经过了详细且适宜自己的设置之后。对于编程软件的设置由于笔者学业不精恕不能详细解释，唯二的劝诫是：

  1. 如果您也是大学生，不建议使用盗版软件，可以利用 JetBrains 或 GitHub Education Pack 等渠道获取正版且有保障的编程环境。
  2. 推荐装一台 Linux 双系统，或索性购买一台 MacBook，Windows 实在不是一个写代码的好地方。

对于视频播放器的配置，笔者个人是一个很小白的赛博仓鼠，随之而来也浅浅对播放器有些了解，在 Windows 平台，并不建议使用广泛好评的 PotPlayer，该软件实际上进阶配置并不方便，且在新版本加入了广告。对于新手用户，个人建议使用开源的 MPV 播放器播放本地视频，但不建议直接使用 GUI 简陋且未做画质优化的原版，而建议使用配置好的 mpv-lazy 项目。

对于进阶用户，可以使用 MPC-BE + LAV Filters + madVR 的组合套餐，达成极为优秀的观影体验，但同时伴随繁琐的配置过程，建议以 VCB-Studio 的教程为基本、萬年冷凍庫的文章为权威参考进行设置。另外请一定容笔者题型：请选择便携版软件进行设置，并在下一次重装系统前做好备份，折腾一次视频播放器设置将耗费很长时间。
